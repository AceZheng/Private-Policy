# A4排版助手技术支持

A4排版助手是一款用于在 iPhone 上整理图片、PDF 页面并导出 A4 PDF 或照片的本地排版工具。

## 主要功能

- 在标准 A4 画布上排版图片和 PDF 页面。
- 从相册或文件导入图片。
- 导入 PDF，并选择一页或多页加入画布。
- 使用 1/2/4/6/8/9/12 宫格等常用模板。
- 拖拽、缩放、旋转、复制和删除画布元素。
- 保存项目，后续继续编辑。
- 导出 A4 PDF 或照片，通过系统分享面板保存或打印。

## 常见问题

### 为什么导出的 PDF 看起来和编辑页面略有差异？

App 会按 A4 物理尺寸生成 PDF。不同预览软件可能会用不同缩放比例显示，请以打印预览中的纸张尺寸为准。

### PDF 多页导入会生成多页 PDF 吗？

当前版本会把选中的 PDF 页面作为多个独立元素排版到同一张 A4 画布上，然后导出为一页 A4 拼版结果。

### 图片和 PDF 会上传到服务器吗？

不会。图片、PDF 和项目文件均在你的设备本地处理。
