# PokerGTO Trainer - 隐私政策和技术支持

本目录包含用于 App Store Connect 的隐私政策和技术支持页面文件。

## 文件

- `index.html` - 首页入口
- `privacy.html` - 隐私政策页面
- `support.html` - 技术支持页面

## 使用说明

1. 将本目录 (`docs/`) 部署到你的 GitHub Pages 或其他静态网页托管服务
2. 在 App Store Connect 中填写：
   - **技术支持 URL**: `https://你的用户名.github.io/仓库名/docs/support.html`
   - **隐私政策 URL**: `https://你的用户名.github.io/仓库名/docs/privacy.html`
3. 记得在 `privacy.html` 和 `support.html` 中填写你的联系邮箱

## 注意事项

- 苹果要求必须提供可访问的隐私政策 URL
- 如果使用 GitHub Pages，只需要把这个 repo 开启 GitHub Pages 即可，源分支选择 main，文件夹选择 `/docs`
