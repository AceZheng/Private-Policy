# A4排版助手 - 技术支持与隐私政策

这个目录可直接用于 GitHub Pages。

## 技术支持

A4排版助手用于在 iPhone 上整理图片和 PDF 页面，并导出 A4 PDF 或照片。

主要功能：

- 标准 A4 画布
- 图片导入与自由排版
- PDF 多页选择导入
- 常用宫格模板
- 元素拖拽、缩放、旋转、复制、删除
- 保存项目，后续继续编辑
- 导出 A4 PDF 或照片
- 通过系统分享面板保存或打印

如需反馈问题，请通过本仓库的 Issues 提交，并说明设备型号、iOS 版本、操作步骤和问题截图。

## 隐私政策

生效日期：2026年6月23日

A4排版助手不会收集、上传、出售或共享你的个人数据。

- 导入的图片和 PDF 仅在设备本地处理。
- 保存的项目文件保存在设备本地 App 文档目录。
- 当前版本不集成第三方统计、广告、追踪或云同步服务。
- 使用系统分享、保存到文件、隔空投送或打印时，数据会交由你主动选择的系统功能或第三方 App 处理。

如果未来版本增加云同步、账号、统计或其他数据功能，本隐私政策会相应更新。
