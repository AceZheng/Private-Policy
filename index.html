技术支持

如果你在使用 A4排版助手 时遇到问题，可以先参考以下说明。

主要功能

  • 在标准 A4 画布上排版图片和 PDF 页面。
  • 从相册或文件导入图片。
  • 导入 PDF，并选择一页或多页加入画布。
  • 使用 1/2/4/6/8/9/12 宫格等常用模板。
  • 拖拽、缩放、旋转、复制和删除画布元素。
  • 保存项目，后续继续编辑。
  • 导出 A4 PDF 或照片，通过系统分享面板保存或打印。

常见问题

为什么导出的 PDF 看起来和编辑页面略有差异？
App 会按 A4 物理尺寸生成 PDF。不同预览软件可能会用不同缩放比例显示，请以打印预览中的纸张尺寸为准。

PDF 多页导入会生成多页 PDF 吗？
当前版本会把选中的 PDF 页面作为多个独立元素排版到同一张 A4 画布上，然后导出为一页 A4 拼版结果。

图片和 PDF 会上传到服务器吗？
不会。图片、PDF 和项目文件均在你的设备本地处理。

隐私政策

生效日期：2026年6月23日

A4排版助手重视你的隐私。本政策说明 App 如何处理你的数据。

1. 我们收集哪些数据

A4排版助手不会收集、上传、出售或共享你的个人数据。

App 中导入的图片、PDF、排版项目和导出文件均在你的设备本地处理。

2. 图片和 PDF 文件

当你选择从相册或文件 App 导入内容时，相关文件仅用于在本机生成排版预览、项目文件、PDF 或照片导出结果。App 不会把这些内容上传到任何服务器。

3. 项目保存

保存项目时，App 会把画布信息和所需素材保存在设备本地的 App 文档目录中。你可以通过删除 App 或系统文件管理方式移除这些数据。

4. 第三方服务

当前版本不集成第三方统计、广告、追踪或云同步服务。

5. 系统分享与打印

当你使用 iOS 系统分享面板、保存到文件、隔空投送或打印功能时，相关数据会交由你选择的系统功能或第三方 App 处理。请同时参考对应服务的隐私政策。

6. 儿童隐私

本 App 不主动收集任何个人信息，也不针对儿童收集数据。

7. 政策更新

如果未来版本增加云同步、账号、统计或其他数据功能，本隐私政策会相应更新。

8. 联系我们

如对隐私政策有疑问，请通过发布本页面的 GitHub 仓库提交 Issue。
